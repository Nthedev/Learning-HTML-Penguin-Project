## 📋 About This Project

A simple informational website about penguins built as my **first HTML practice project** while learning web development fundamentals. This demonstrates my journey from basic HTML structure to proper document anatomy.

> 💡 **My Learning Journey:** I built this initially without knowing proper document structure (`<head>`, `<title>`). I've since learned best practices, fixed typos, added an image with alt text, and updated the code to follow HTML standards — all while tracking this evolution transparently.

---

## 🛠️ What Changed Between Versions

| Element | Original (v1.0) | Updated (Current) |
|---------|-----------------|-------------------|
| `<!DOCTYPE html>` | ✅ Yes | ✅ Yes |
| `<html>` root tag | ❌ No | ✅ Yes |
| `<head>` section | ❌ No | ✅ Yes |
| `<title>` tag | ❌ No | ✅ Yes (`<title>Penguins</title>`) |

## 📜 Detailed Version History

### v1.0 (Original Build)
| Issue | Problem |
|-------|---------|
| **Structure** | No `<html>` root tag, no `<head>` section, no `<title>` |
| **Typos** | "practise" → "practice", "commited" → "committed", "tounge" → "tongue", "pray" → "prey", "Mascott" → "Mascot" |
| **Content** | No images, limited source attribution |

### v1.1-current (Updated Version)
| Improvement | Solution |
|-------------|----------|
| **Document Structure** | Added `<html>` root tag, `<head>` section with `<title>` |
| **Typos** | Fixed all 5 spelling errors for professional quality |
| **Accessibility** | Added `<img>` with descriptive `alt` text for screen readers |
| **Attribution** | Added IUCN Red List citation alongside Wikipedia |

---

## 🛠️ What's Currently Implemented

| Element Type | Usage |
|--------------|-------|
| Document Type | `<!DOCTYPE html>` |
| Root Element | `<html>` (opening and closing tags) |
| Head Section | `<head>` with `<title>` tag |
| Body | `<body>` wrapper |
| Headings | `<h1>`, `<h2>`, `<h3>` |
| Paragraphs | `<p>` |
| Lists | `<ul>`, `<li>` |
| Emphasis | `<strong>`, `<em>` |
| Line Breaks | `<br />` |
| Entity Codes | `&#128075;` (wave emoji) |
| Images | `<img>` with `src` and `alt` attributes |

**Still learning:** `<meta charset="UTF-8">`, `<meta name="viewport">`, CSS styling

## 📄 Content Structure

The site includes:
- Introduction and purpose statement
- General information about penguins (habitat, diet)
- Featured penguin image with alt text (Adélie Penguins on Iceberg)
- Featured species (Emperor, Little Blue)
- Conservation status overview
- Fun facts section
- Source attributions (Wikipedia + IUCN Red List)
