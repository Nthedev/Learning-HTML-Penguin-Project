## 📋 About This Project

A simple informational website about penguins built as a **practice project** for learning fundamental HTML5 skills. This demonstrates my current progress in web development basics.

> Note: This is intentionally kept minimal—using only basic HTML without CSS styling—to focus on learning proper structure and semantic elements first.

---

## 🛠️ What's Actually Implemented

| Element Type | Usage |
|--------------|-------|
| Document Type | `<!DOCTYPE html>` |
| Headings | `<h1>`, `<h2>`, `<h3>` |
| Paragraphs | `<p>` |
| Lists | `<ul>`, `<li>` |
| Emphasis | `<strong>`, `<em>` |
| Line Breaks | `<br />` |
| Entity Codes | `&#128075;` (wave emoji) |

---

## 📄 Content Structure

The site includes:
- Introduction and purpose statement
- General information about penguins (habitat, diet)
- Featured species (Emperor, Little Blue)
- Conservation status overview
- Fun facts section

All content compiled from [Wikipedia](https://en.wikipedia.org/wiki/Penguin) as a learning source.
